# ENet-SAD_Pytorch
 Pytorch implementation of "Learning Lightweight Lane Detection CNNs by Self Attention Distillation (ICCV 2019)"

### Requirements
* pytorch
* tensorflow (for tensorboard)
* opencv
* numpy
* scipy
